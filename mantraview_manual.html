MantraView 사용설명서

MantraView는 손을 대지 않고 목소리만으로 모바일 화면을 자유롭게 스크롤하고 제어할 수 있는 스마트 접근성 솔루션입니다.

본 사용설명서는 MantraView의 모든 제어 모드와 각 모드별 특화 기능, 지원 음성 명령어를 안내합니다.

💡 최초 실행 권한 안내

MantraView가 다른 앱 화면을 제어하기 위해 접근성 권한 및 마이크 권한이 반드시 필요합니다. 최초 실행 시 나타나는 접근성 허용 안내에 따라 MantraView 서비스를 활성화해 주세요.

MantraView User Manual

MantraView is a smart accessibility solution that lets you scroll and control your mobile screen hands-free, using only your voice.

This manual covers all control modes, specific features, and supported voice commands for MantraView.

💡 Permissions Required

MantraView requires Accessibility Services and Microphone permissions to perform gestures. Please enable the MantraView service in accessibility settings during the first launch.

🎮 4가지 핵심 모드

MantraView는 사용 환경에 맞게 선택할 수 있는 4가지 화면 제어 방식을 지원합니다.

🎙️
음성 인식 모드(*동영상에서는 소리가 끊김. 사용제한)
"내려", "올려", "멈춰", "클릭" 등 직관적인 음성 명령어로 기기를 완전히 음성으로 제어합니다.
📖
읽기 모드 (OCR & TTS)
화면 상의 텍스트를 AI로 판독(OCR)하여 음성(TTS)으로 읽어줍니다. 다이얼형 사각 틀로 읽을 영역을 자유롭게 지정할 수 있습니다.
🔘
상하 버튼 모드
화면에 떠 있는 플로팅 방향 버튼을 직접 터치하여 연속 스크롤, 정지, 속도 조절 등을 수동으로 제어합니다.
⚡
슬라이드 모드
화면을 휙 쓸어넘기는 슬라이드 제스처를 시뮬레이션하여 사진이나 카드형 컨텐츠를 감상할 때 편리합니다.

🎮 4 Core Modes

MantraView supports four screen control modes tailored to your needs.

🎙️
Voice Control Mode(*The video continues to stop sounding; use restricted)
Control your device fully with simple voice commands like "Down", "Up", "Stop", and "Click".
📖
Read Mode (OCR)
Extracts text from the screen using AI (OCR) and reads it aloud (TTS). Set custom reading boundaries with the resizable target frame.
🔘
Manual Button Mode
Directly tap the floating overlay buttons to start continuous scrolling, stop, or adjust speeds.
⚡
Slide Mode
Simulates quick swipe-up/down gestures, optimized for scrolling through video feeds like Reels, Shorts, and TikTok.

🗣️ 전체 지원 음성 명령어

언어 설정(한국어/영어 등)에 맞는 발음으로 편리하게 스크롤 및 기능을 제어해 보세요.

기능 구분 대표 명령어 (한국어 예시) 기능 설명
스크롤 이동 내려 / 올려 화면 아래 또는 위 방향으로 스크롤을 이동합니다.
자동 스크롤 시작 / 정지 (멈춰) 지속적으로 부드럽게 스크롤하는 자동 스크롤을 토글합니다.
이동 속도 빨리 / 천천히 자동 스크롤 속도를 빠르게(속도 업) 또는 느리게(속도 다운) 변경합니다.
경계 바로가기 처음으로 / 끝으로 화면의 가장 위(최상단) 또는 가장 아래(최하단)로 즉시 이동합니다.
방향 슬라이드 왼쪽 (좌) / 오른쪽 (우) 좌/우 페이지 넘김 슬라이드 액션을 시뮬레이션합니다.
기본 터치 클릭 화면 한가운데 정중앙 영역을 가상 터치합니다.
지능형 OCR 터치 [숫자/글자] + 클릭 예: "3번 클릭" 또는 "닫기 클릭"을 입력 시 해당 글자가 있는 화면 위치를 자동 판독하여 터치합니다.
페이지 이동 다음페이지 (다음 장) 화면 하단에 페이지네이션 버튼이 감지되면 클릭하고, 없을 시 슬라이드로 화면을 좌측으로 밀어 넘깁니다.
뒤로가기 뒤로 이전 화면으로 가거나 웹 브라우저에서 뒤 페이지로 이동합니다.
시스템 볼륨 볼륨 올려 / 볼륨 내려 미디어 볼륨 단계를 한 단계 키우거나 줄입니다. ("소리 키워/줄여", "볼륨 업/다운" 호환)
광고/팝업 닫기 닫기 / 광고 닫기 / x 화면 내의 확인, 닫기, 건너뛰기, Skip, ❌ 버튼을 인식하여 즉시 강제 종료합니다.
읽기 모드 제어 읽기 시작 / 읽기 중단 화면 속 글자를 소리 내어 낭독하는 TTS 엔진을 구동하거나 중단시킵니다.
영역 설정 제어 영역 설정 / 영역 닫기 읽기 모드의 판독 사각틀 범위를 조정하는 모드로 들어가거나 완료합니다.
제어 모드 변경 음성 인식 모드 / 읽기 모드 플로팅 제어기 모드를 신속하게 음성으로 상호 변환합니다. ("상하 버튼 모드", "슬라이드 모드" 포함)
제어기 종료/취소 앱종료 / 취소 화면에서 플로팅 버튼을 제거하고 서비스를 종료하거나, 진행 중인 동작/창을 취소합니다.
공지/초기화 공지사항 / 초기화 업데이트 알림 공지를 확인하거나 누적된 환경 설정을 초기화합니다.

🗣️ Supported Voice Commands

Simply speak clearly in your configured language. Standard and English commands are shown below.

Category Voice Command Examples Action Description
Scroll Direction Down / Up Scrolls the screen down or up.
Auto-Scroll Start / Stop Starts or stops continuous smooth scrolling.
Speed Control Faster / Slower Increases or decreases auto-scroll speed.
Jump to Edge To Start / To End Jumps directly to the very top or very bottom of the screen.
Side Swiping Left / Right Simulates side-swiping gestures (page transitions).
Center Click Click Simulates a tap on the exact center of your screen.
AI Target Click [Number/Text] + Click E.g. "Three Click" or "Close Click" will analyze screen text via OCR and tap the exact target coordinates.
Pagination Next Page Clicks pagination buttons (e.g. next page numbers, arrows) or swipes left.
Navigation Back / Forward Navigates backward or forward on screen stack or web pages.
Media Volume Volume Up / Volume Down Increases or decreases system media volume. ("Louder", "Lower volume" compatible)
Ad & Dialog Dismiss Close / Close Ad / x Identifies buttons like OK, Cancel, Skip, Close, or ❌ via OCR and clicks them.
Reader Controls Start Reading / Stop Reading Starts or halts the screen reader TTS speech engine.
Range Frame Range Settings / Close Range Enters or exits text selection box layout configuration.
Change Control Mode Voice Mode / Read Mode Instantly switches between active overlay configurations. ("Manual Mode", "Slide Mode" included)
Overlay Manager Exit App / Cancel Closes settings panels, stops continuous scroll, or hides the floating controller.
Notice / Reset Notice / Reset Views announcements or resets settings to defaults.

🎨 스크롤 프리셋 & 다크/라이트 테마

감상하는 콘텐츠 타입에 맞춰 속도 프로필을 빠르게 불러올 수 있습니다.

📋 스크롤 프리셋 종류

  • 일반 (Default): 웹 서핑, 피드 스크롤을 위한 일반 제어 감도 제공.
  • 웹툰 (Webtoon): 위아래로 연속 스크롤 시 정지 구간을 없애고 100% 매끄러운 수직 연속 이동 적용.
  • 뉴스 (News): 텍스트를 장시간 자세히 읽을 수 있도록 스크롤 간격이 짧아지고 정지 시간이 늘어납니다.
  • 릴스 (Reels): 동영상이 끊기지 않는 자동 스크롤과 시간설정 변경 제공.

🌗 테마 선택 (프리셋/테마 화면에서 설정 가능)

플로팅 버튼 디자인을 어두운 배경에 걸맞는 세련된 다크 테마 (화이트 글자 포함)와 밝은 배경에 적합한 가독성 높은 밝은 테마 (다크 글자 포함) 중에서 설정하고 실시간으로 반영할 수 있습니다.

🎨 Scroll Presets & Dark/Light Themes

Quickly apply scroll speed profiles matching the exact type of content you are enjoying.

📋 Scroll Presets

  • Default: Standard scroll speeds for daily web surfing and social feeds.
  • Webtoon: Disables pauses and increases scrolling consistency for seamless vertical webtoon reading.
  • News: Extends pause times and shortens distances to read articles comfortably.
  • Reels: Provides automatic scrolling and changing time settings that don't break the video

🌗 Themes (Configurable via Preset/Theme tab)

Toggle the overlay visuals in real time. Choose from a sleek Dark Theme (White Text) or a high-contrast Light Theme (Dark Text) tailored for daytime visibility.

🔊 주변 소음 극대화 대응 팁

MantraView는 오프라인 기기 내장 음성인식 엔진(On-Device)과 지능형 소음 감쇄 모듈을 결합하여, 시끄러운 환경에서도 강력한 인식 성능을 보장합니다.

  • 대기 타임아웃 튜닝: 소음으로 인한 오인식을 막고자 음성 묵음 판단 시간을 여유롭게 3.0초 이상으로 자동 고정하고, 마이크 재활성화 속도를 보정했습니다.
  • 권장 발성: 단어 사이에 너무 긴 텀을 두지 말고 "내려", "정지"처럼 끝마무리를 명확하게 발음하면 주변 잡음이 섞여도 높은 확률로 필터링되어 동작합니다.

🔊 Tips for Noise-Robust Performance

MantraView utilizes on-device speech processing models to prevent background noise from interfering with your commands.

  • Extended Timeouts: Silence detection thresholds are set to 3.0+ seconds, ensuring smooth recognition without constant mic re-open cycles.
  • Speaking Style: Pronounce target actions clearly and concisely (e.g. "Down", "Stop") for maximum noise filter efficiency.
