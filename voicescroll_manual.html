🚀 VoiceScroll 사용설명서 (v1.2.3)

VoiceScroll은 음성 명령을 통해 스마트폰 화면을 손대지 않고 제어할 수 있는 혁신적인 접근성 도구입니다. 웹툰, 뉴스, SNS 등을 이용할 때 최상의 편의성을 제공합니다.

1. 시작하기

  • 서비스 활성화: 앱 우측 하단의 [서비스 활성화] 버튼을 눌러 안드로이드 설정에서 VoiceScroll 접근성 서비스를 '사용 중'으로 변경하세요.
  • 버튼 표시: 메인 화면의 [시작] 버튼을 누르면 화면에 파란색 플로팅 버튼이 나타납니다.
  • 음성 인식 시작: 플로팅 버튼을 한 번 터치하면 마이크가 활성화되며 음성 명령을 대기합니다. *동영상에서 안드로이드 기본 음성 인식(Google SpeechRecognizer)의 시스템적인 한계 때문에 음성인식 모드는 소리가 끊기는 현상이 발생합니다. 동영상에서는 사용을 자제해주세요.

2. 기본 음성 명령어

  • 내려 / 올려: 화면을 1회 살짝 스크롤합니다.
  • 시작: 아래 방향으로 자동 스크롤을 무한히 시작합니다.
  • 멈춰 / 정지: 모든 스크롤 동작을 즉시 중단합니다.
  • 빨리 / 천천히: 스크롤이 움직이는 속도를 조절합니다.
  • 처음으로 / 끝으로: 화면의 맨 위 또는 맨 아래로 즉시 이동합니다.
  • 뒤로: 시스템 '뒤로가기' 동작을 수행합니다.
  • 클릭: 화면의 정중앙을 1회 터치합니다.

3. 프리미엄 기능 (Premium)

  • 광고 제거: 앱 내의 모든 배너 및 전면 광고가 사라집니다.
  • 자동 실행 앱 설정: 특정 앱(예: 크롬, 카카오페이지)을 열 때 보이스스크롤이 자동으로 켜지도록 설정할 수 있습니다.
  • 커스텀 명령어: "내려" 대신 "넥스트", "고" 등 본인이 원하는 단어로 명령어를 바꿀 수 있습니다.
  • 모드 전환: 순환버튼으로 웹툰모드, 뉴스모드, 릴스모드를 즉시 최적화된 설정으로 적용할 수 있습니다.
  • 정밀 튜닝: 각 모드별 스크롤 속도, 이동 거리, 대기 시간을 0.1초 단위로 세밀하게 조절 가능합니다.

4. 꿀팁 및 주의사항

  • 24시간 무료 체험: 광고 30초 시청 시 24시간 동안 모든 프리미엄 기능을 무료로 이용할 수 있습니다.
  • 정지 버튼 길게 누르기: 플로팅 버튼 중 정지(■) 버튼을 길게 누르면 보이스스크롤 서비스가 즉시 종료됩니다.
  • 안전 범위: 시스템 상단 바와의 충돌을 방지하기 위해 화면의 중앙 80% 영역 내에서 제스처가 작동합니다.

🚀 VoiceScroll User Manual (v1.2.3)

VoiceScroll is an innovative accessibility tool that allows you to control your smartphone screen hands-free via voice commands. It provides maximum convenience when using webtoons, news, SNS, etc.

1. Getting Started

  • Activate Service: Press the [Activate Service] button at the bottom right of the app and change the VoiceScroll accessibility service to 'On' in Android settings.
  • Show Button: Press the [Start] button on the main screen, and a blue floating button will appear on the screen.
  • Start Voice Recognition: Touch the floating button once to activate the microphone and wait for voice commands. *Due to the system limitations of Android's default voice recognition (Google SpeechRecognizer), the voice recognition mode may cause audio stuttering in videos. Please refrain from using it while watching videos.

2. Basic Voice Commands

  • Down / Up: Scrolls the screen slightly once.
  • Start: Starts auto-scrolling downwards infinitely.
  • Stop: Immediately stops all scrolling actions.
  • Faster / Slower: Adjusts the scrolling speed.
  • To Start / To End: Instantly jumps to the top or bottom of the screen.
  • Back: Performs the system 'Back' action.
  • Click: Touches the exact center of the screen once.

3. Premium Features (Premium)

  • Remove Ads: All banner and interstitial ads within the app will be removed.
  • Auto-Start Apps: You can set VoiceScroll to turn on automatically when you open specific apps (e.g., Chrome, KakaoPage).
  • Custom Commands: You can change the commands to words you prefer, such as "Next" or "Go" instead of "Down".
  • Mode Switching: Use the cycle button to instantly apply optimized settings for Webtoon Mode, News Mode, and Reels Mode.
  • Fine Tuning: You can finely adjust the scroll speed, movement distance, and pause time for each mode in 0.1-second increments.

4. Tips & Precautions

  • 24-Hour Free Trial: Watch a 30-second ad to use all premium features for free for 24 hours.
  • Long Press Stop Button: Long press the Stop (■) button on the floating menu to instantly exit the VoiceScroll service.
  • Safe Area: To prevent conflicts with the system top bar, gestures operate within the center 80% area of the screen.
